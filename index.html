Copy-Past respects your privacy. This Privacy Policy explains how we handle user data and privacy.

Data Collection and Usage

  • No Data Collection: We do not collect, store, use, or share any personal data or browsing data from users of our extension.
  • Permissions: The permissions requested by the extension (activeTab, storage) are solely for the functionality of the extension. They are used to:
    • activeTab: Allow the extension to interact with the currently active tab to dynamically update content based on user preferences.
    • storage: Store user preferences regarding which HTML tags should have the "Copy" button enabled.
  • No Tracking: We do not use any tracking technologies such as cookies, web beacons, or similar mechanisms.

User Control

Users have full control over the functionality of the extension through the popup interface, where they can enable or disable the "Copy" button for specific HTML tags.

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
